Menginstall Xfce 4.12 Di Fedora 21 Workstation

@alunux · Apr 9, 2015 · 1 min read

Bismillahirrahmanirrahim. [Chapter 1] Instalasi Paket-Paket Xfce 4.12 Dalam chapter ini, semua tahap yang dilakukan akan memakai akun root. Buka terminal Anda, lalu ketikkan su - Sekarang masuk ke direktori /etc/yum.repos.d/. Kita akan mendownload file .repo dari Copr. Perintah: cd /etc/yum.repos.d/ && wget https://copr.fedoraproject.org/coprs/nonamedotc/xfce412/repo/fedora-21/nonamedotc-xfce412-fedora-21.repo Install paket-paket Xfce 4.12. Di sini kita akan menginstall Xfce minimal, jadi aplikasi seperti Task Manager, Catfish, dan paket tambahan lainnya tidak akan terinstall.
