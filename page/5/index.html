Install Driver Printer Canon PIXMA MP-237 Di Fedora 20 64 bit

@alunux · Jul 20, 2014 · 1 min read

Bismillahirrahmanirrahim. Download paket drivernya terlebih dahulu di website resmi Canon. Klik Di Sini. Buka terminal dan login sebagai root. Command: su - Buka aplikasi text editor (misal gedit/kwrite/kate/vim/nano/lainnya). Copy paste tulisan dibawah ini ke text editor: Name: libtiff_so3_link Version: 1.0 Release: 1%{?dist} Summary: Symlink from libtiff.so.5 to libtiff.so.3 License: GPL Requires: %{_libdir}/libtiff.so.5 %ifarch x86_64 Provides: libtiff.so.3()(64bit) %else Provides: libtiff.so.3 %endif %description %{summary} %prep %build %install ln -s %{_libdir}/libtiff.

Berkenalan Dengan Fedora Spins

@alunux · Jul 20, 2014 · 4 min read

Apakah spin itu? Fedora spin adalah versi alternatif dari Fedora, disesuaikan untuk berbagai jenis pengguna dengan memilih aplikasinya sendiri dan kostumisasi yang lain. Spin pertama tersedia bersamaan dengan rilis Fedora 7 pada Mei 2007. Proses pembuatan Spin dan perbaikan Spin dikelola oleh Fedora Spins SIG. (Sumber: http://spins.fedoraproject.org/id/about) Kalau Anda adalah pengguna Ubuntu maka Anda pastinya tidak asing dengan berbagai varian Ubuntu, seperti Kubuntu, Ubuntu GNOME, Xubuntu, Lubuntu, dan Edubuntu. Fedora Spins diciptakan untuk memudahkan pekerjaan dengan seseorang dengan kebutuhan spesifik.
