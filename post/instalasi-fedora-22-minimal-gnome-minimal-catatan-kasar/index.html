Instalasi Fedora 22 Minimal + GNOME Minimal (Catatan Kasar)

@alunux · Aug 18, 2015 · 2 min read

Bismillahirrahmanirrahim.

[Chapter 1] Sebelum Instalasi

  1. Download paket NetworkManager-wifi untuk mengakses internet via jaringan wifi: Download 64-bit Download 32-bit

  2. Download paket wpa-supplicant sebagai dependensi paket NetworkManager-wifi: Download 64-bit Download 32-bit

  3. Simpan kedua paket tersebut.

[Chapter 2] Instalasi

[Chapter 3] Koneksi ke Wifi

  1. Mount partisi di mana paket NetworkManager-wifi dan wpa_supplicant berada. Saya menyimpannya di paritisi /dev/sda2, sudo mount /dev/sda2 /mnt

  2. Masuk ke direktori /mnt, cd /mnt

  3. Lakukan instalasi kedua paket tadi. Untuk 32-bit, sudo dnf –disablerepo=* install NetworkManager-wifi-1.0.2-1.fc22.i686.rpm wpa_supplicant-2.3-3.fc22.i686.rpm

  4. Untuk 64-bit, sudo dnf –disablerepo=* install NetworkManager-wifi-1.0.2-1.fc22.x86_64.rpm wpa_supplicant-2.3-3.fc22.x86_64.rpm

  5. Unmount partisi, sudo umount -Rf /mnt

  6. sudo systemctl restart NetworkManager

  7. sudo nmcli radio wifi on

  8. sudo ifconfig wlp3s0 up

  9. iwlist wlp3s0 scan

  10. nmcli dev wifi list

  11. nmcli device wifi rescan

  12. sudo nmcli device wifi connect PTIIK

  13. ping google.com

[Chapter 4] Instal GNOME Minimal

  1. sudo dnf update

  2. sudo dnf install gnome-shell gnome-js-common gnome-terminal mesa-dri-drivers xorg-x11-drivers

  3. sudo rm /etc/systemd/system/default.target

  4. sudo systemctl –force enable gdm

Catatan:

Saya sengaja membuat catatan kasar ini sebagai pengingat apabila saya akan menginstal ulang Fedora di laptop saya. Fedora Minimal ini mirip seperti Arch Linux. Berhubung saya menyukai Arch Linux, akhirnya saya mencari cara agar Fedora yang terinstal di laptop saya bersifat minimalis. Dengan instalasi yang minimalis ini, saya dapat membangun sistem yang saya inginkan layaknya sedang berada di Arch Linux. Tidak direkomendasikan bagi pemula yang baru memakai Linux dan belum tahu dasar sistem manajemen paket.

Tulisan ini dikerjakan menggunakan WordPress dengan bantuan Google Chrome 44.0.2403.155. Sistem operasi yang digunakan yaitu Fedora 22. Semua percobaan di atas dilakukan dalam mesin virtual dengan Fedora 22 Minimal terinstal didalamnya. Virtualisasi dilakukan dengan bantuan VirtualBox 4.3.30. Semua screenshot yang ada di tulisan ini diambil dengan menggunakan fitur screenshot yang ada di VirtualBox.
