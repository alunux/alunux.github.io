Menginstall Xfce 4.12 Di Fedora 21 Workstation

@alunux · Apr 9, 2015 · 1 min read

Bismillahirrahmanirrahim.

[Chapter 1] Instalasi Paket-Paket Xfce 4.12

  1. Dalam chapter ini, semua tahap yang dilakukan akan memakai akun root. Buka terminal Anda, lalu ketikkan su -

  2. Sekarang masuk ke direktori /etc/yum.repos.d/. Kita akan mendownload file .repo dari Copr. Perintah: cd /etc/yum.repos.d/ && wget https://copr.fedoraproject.org/coprs/nonamedotc/xfce412/repo/fedora-21/nonamedotc-xfce412-fedora-21.repo

  3. Install paket-paket Xfce 4.12. Di sini kita akan menginstall Xfce minimal, jadi aplikasi seperti Task Manager, Catfish, dan paket tambahan lainnya tidak akan terinstall. Oh iya, install dulu paket kernel-modules-extra. Ia diperlukan oleh paket xl2tpd. Perintah: yum install -y kernel-modules-extra. Kemudian instal paket-paket Xfcenya. Perintah: yum install -y @xfce-desktop

  4. Tunggu sampai proses instalasi selesai dengan sempurna. Tenang, paket yang akan diinstal tidaklah banyak.
