Install Driver Printer Canon PIXMA MP-237 Di Fedora 20 64 bit

@alunux · Jul 20, 2014 · 1 min read

Bismillahirrahmanirrahim.

  1. Download paket drivernya terlebih dahulu di website resmi Canon. Klik Di Sini.

  2. Buka terminal dan login sebagai root. Command: su -

  3. Buka aplikasi text editor (misal gedit/kwrite/kate/vim/nano/lainnya). Copy paste tulisan dibawah ini ke text editor:

        Name:           libtiff_so3_link
    Version:        1.0
    Release:        1%{?dist}
    Summary:        Symlink from libtiff.so.5 to libtiff.so.3
    License:        GPL
    Requires:       %{_libdir}/libtiff.so.5
    %ifarch x86_64
    Provides:       libtiff.so.3()(64bit)
    %else
    Provides:       libtiff.so.3
    %endif
    %description
    %{summary}
    %prep
    %build
    %install
    ln -s %{_libdir}/libtiff.so.5   %{buildroot}/%{_libdir}/libtiff.so.3
    mkdir -p %{buildroot}/%{_libdir}/
    %files
    %{_libdir}/libtiff.so.3
    %changelog
    * Thu Mar 06 2014  1.0-1
    - Initial build.

  4. Save dengan nama libtiff_so3_link.spec

  5. Kembali ke terminal. Sekarang kita buat paket rpm libtiff_so3_link. Command: rpmbuild -bb libtiff_so3_link.spec

  6. Install paket libtiff_so3_link. Paket tersebeut terletak di /root/rpmbuild/RPMS/x86_64. Command: yum install rpmbuild/RPMS/x86_64/libtiff_so3_link-1.0-1.fc20.x86_64.rpm

  7. Ekstrak paket driver yang telah didownload tadi.

  8. Masuk ke lokasi folder ekstrakan lewat terminal. Misal foldernya berada di folder /home/alunux/Downloads/cnijfilter-mp230series-3.80-1-rpm/. Command: cd /home/alunux/Downloads/cnijfilter-mp230series-3.80-1-rpm/

  9. Install driver. Command: ./install.sh

  10. Jawab semua pertanyaan yang muncul saat proses instalasi berjalan. Jika sudah selesai, sekarang Anda mencoba printer tersebut.

Cara ini berhasil di laptop saya.

OS : Fedora 20 64 bit
Kernel : 3.14.9-200.fc20.x86_64
