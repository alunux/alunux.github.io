Yang Saya Lakukan setelah Instal Fedora 25 Workstation

@alunux · Nov 27, 2016 · 4 min read

Bismillahirrahmanirrahim.

Spesifikasi laptop yang saya gunakan:

  • ASUS A455L
  • RAM 2 GB
  • Prosesor Intel Core i3-4030 1,9 GHz
  • Kapasitas HDD 500 GB
  • VGA Dual Graphics, Intel HD Graphics 4400 + Nvidia Geforce 820M 2 GB
  • Wireless Adapter Atheros
  • Ethernet Realtek
  • Bluetooth 4.0
  • Slot 1x USB 3.0, 2x USB 2.0, HDMI, LAN, SD Card Reader, 1x Combo Audio Jack
  • HD WebCam

[Chapter 1] Wajib Bagi Saya

  1. Mengatur konfigurasi DNF agar DNF menyimpan semua cache yang pernah Ia gunakan dan mencari mirror tercepat dengan menggunakan fastestmirror. Buka terminal dan jalankan perintah su -c ‘gedit /etc/dnf/dnf.conf’ dan tambahkan keepcache=1 dan fastestmirror=1 pada baris baru. Terakhir simpan berkas tersebut.

  2. Hapus aplikasi yang tidak diperlukan. Ada dua cara yang dapat dilakukan untuk menghapus aplikasi, yaitu menggunakan perintah su -c ‘dnf remove aplikasi’ yang dijalankan melalui terminal dan menggunakan GNOME Software yang dapat diakses di Dash milik GNOME Shell.

  3. Perbarui sistem. Ada dua cara yang dapat dilakukan untuk memperbarui sistem, yaitu menggunakan perintah su -c ‘dnf upgrade’ (jalankan su -c ‘dnf distro-sync’ jika Anda telah menggunakan versi Alpha, Beta, atau RC) yang dijalankan melalui terminal dan menggunakan GNOME Software yang dapat diakses di Dash milik GNOME Shell.

  4. Menambahkan repositori RPM Fusion, buka terminal dan jalankan perintah su -c ‘dnf install http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm’

  5. Instal Multimedia Codecs untuk keperluan multimedia. Buka terminal dan jalankan perintah su -c ‘dnf install gstreamer1-libav gstreamer1-plugins-bad-free gstreamer1-plugins-bad-freeworld gstreamer1-plugins-base gstreamer1-plugins-good gstreamer1-plugins-ugly gstreamer-ffmpeg gstreamer-plugins-bad gstreamer-plugins-bad-free gstreamer-plugins-bad-nonfree gstreamer-plugins-base gstreamer-plugins-espeak gstreamer-plugins-fc gstreamer-plugins-good gstreamer-plugins-ugly gstreamer-rtsp faac faad2 flac lame libdca libmad libmatroska x264 x265 xvidcore’

  6. Instal aplikasi pemaketan berkas. Buka terminal dan jalankan perintah su -c ‘dnf install cabextract lzip p7zip p7zip-plugins’

  7. Mengatur shortcut pada keyboard. Buka Settings -> Keyboard -> Shortcuts.

[Chapter 2] Menyesuaikan Desktop

  1. Install GNOME Tweak Tool, dconf Editor, dan GNOME Tracker Preferences. Buka terminal dan jalankan su -c ‘dnf install gnome-tweak-tool dconf-editor tracker-preferences’

  2. Menambahkan Extension pada GNOME Shell. Buka Firefox dan pergi ke halaman http://extensions.gnome.org/. Ketika Anda membuka laman tersebut akan muncul notifikasi dari Firefox, klik tombol Allow and Remember pada notifikasi tersebut. Muat ulang lamannya dan silahkan jelajahi Extension yang Anda inginkan.

  3. Mengubah ukuran Header Bar/Title Bar. Tekan tombol Alt + F2 pada keyboard. Isikan gedit .config/gtk-3.0/gtk.css pada kotak isian yang muncul dan tekan tombol enter pada keyboard. Ketika jendela Gedit telah terbuka, isikan dengan:

    kemudian simpan berkas tersebut (Ctrl + S) dan tutup Gedit. Logout kemudian Login kembali untuk melihat perubahan.

  4. Instal Qt4 Config dan Qt5 Settings untuk mengatur tampilan aplikasi yang menggunakan Qt sebagai GUI Toolkitnya. Buka terminal dan jalankan perintah su -c ‘dnf remove qgnomeplatform && dnf install qt-config qt5ct’

  5. Instal Fedy. Buka terminal dan jalankan perintah bash -c ‘su -c “curl http://folkswithhats.org/fedy-installer -o fedy-installer && chmod +x fedy-installer && ./fedy-installer”’

  6. Instal tema lain http://gnome-look.org

[Chapter 3] Instalasi Driver Dual Graphics Intel + Nvidia

Buka terminal dan jalankan perintah-perintah di bawah ini secara sistematis.

  1. su -c ‘dnf -y –nogpgcheck install http://install.linux.ncsu.edu/pub/yum/itecs/public/bumblebee/fedora25/noarch/bumblebee-release-1.2-1.noarch.rpm’ su -c ‘dnf -y –nogpgcheck install http://install.linux.ncsu.edu/pub/yum/itecs/public/bumblebee-nonfree/fedora25/noarch/bumblebee-nonfree-release-1.2-1.noarch.rpm’

  2. Untuk Fedora versi 32-bit: su -c ‘dnf install bumblebee-nvidia bbswitch-dkms primus kernel-devel’

  3. Untuk Fedora versi 64-bit: su -c ‘dnf install bumblebee-nvidia bbswitch-dkms VirtualGL.x86_64 VirtualGL.i686 primus.x86_64 primus.i686 kernel-devel’

  4. Hidupkan ulang komputer anda

  5. Coba buka Nvidia X Server Settings untuk melihat hasilnya

  6. Pasang Extension https://extensions.gnome.org/extension/985/optirun/ pada GNOME Shell untuk memudahkan menjalankan aplikasi dengan vga Nvidia

  7. Informasi Lanjut https://fedoraproject.org/wiki/Bumblebee

[Chapter 4] Menghemat Baterai Laptop dan Tombol fn untuk Kecerahan Layar

  1. Instal TLP. Buka terminal dan jalankan perintah su -c ‘dnf install tlp’. Setelah instalasi selesai, jalankan perintah su -c ‘tlp start’

  2. Tambahkan beberapa parameter kernel. Buka terminal dan jalan perintah su -c ‘gedit /etc/default/grub’. Pada baris GRUB_CMDLINE_LINUX, tambahkan “i915.i915_enable_rc6=1 i915.i915_enable_fbc=1 i915.lvds_downclock=1 acpi_osi=”. Contoh punya saya: GRUB_CMDLINE_LINUX=“nouveau.modeset=0 rd.driver.blacklist=nouveau rhgb quiet i915.i915_enable_rc6=1 i915.i915_enable_fbc=1 i915.lvds_downclock=1 acpi_osi=”. Kemudian simpan berkas tersebut dan jalankan perintah su -c ‘grub2-mkconfig -o /boot/grub2/grub.cfg’.

[Chapter 5] Daftar Aplikasi Pilihan

  • GIMP (dnf install gimp)
  • Inkscape (dnf install inkscape)
  • Gthumb (dnf install gthumb)
  • GNOME Builder (dnf install gnome-builder)
  • Deluge (dnf install deluge)
  • VLC (dnf install vlc)
  • MPV (dnf install mpv)
  • Shutter (dnf install shutter)
  • Youtube-DL (dnf install youtube-dl)
  • Wine (dnf install wine)
  • GNOME Logs (dnf install gnome-logs)
  • Bleachbit (dnf install bleachbit)
  • Terminix (dnf copr enable heikoada/terminix && dnf install terminix)
  • Yumex-DNF (dnf install yumex-dnf)
  • Chromium (dnf install chromium)
  • Google Chrome (https://www.google.com/chrome/browser/desktop/index.html)
