Mengganti Alamat DNS Server di Fedora 23

@alunux · Jun 2, 2016 · 1 min read

Bismillahirrahmanirrahim.

  1. Buka GNOME Control Settings
  2. Pilih menu Network
  3. Pilih koneksi yang ingin diubah pengaturan DNSnya
  4. Saat jendela pengaturan terbuka, pilih menu IPv4 yang terletak di panel sebelah kiri
  5. Matikan fitur Automatic pada bagian DNS dan isikan alamat DNS server yang diinginkan pada kotak isian server
  6. Klik tombol Apply untuk menyimpan konfigurasi yang telah dilakukan
  7. Kita perlu menjalankan ulang service NetworkManager untuk menerapkan konfigurasi yang telah dilakukan. Buka terminal kemudian jalankan perintah sudo systemctl restart NetworkManager
  8. Selesai

Tulisan ini dikerjakan menggunakan WordPress dengan bantuan Google Chrome 51.0.2704.79. Sistem operasi yang digunakan yaitu Fedora 23. Versi GNOME yang digunakan adalah 3.20. Alamat DNS Server diberikan oleh proyek OpenDNS. Semua screenshot yang ada di tulisan ini diambil dan disunting dengan Shutter 0.93.1 Rev.1278.
