Tugas Tantangan 1 [Sistem Operasi]

@alunux · Sep 18, 2014 · 2 min read

Bismillahirrahmanirrahim.

Postingan ini bertujuan untuk memenuhi tugas praktikum mata kuliah Sistem Operasi. Adapun tantangan yang diberikan:

  1. Screenshot login screen dengan nama user masing-masing,
  2. instal aplikasi tambahan melalui terminal, screenshot syntax, proses dan hasil,
  3. jalankan aplikasi Line dari windows screenshot fullscreen.

[Chapter 1] Mengambil Screenshot Login Screen

Untuk mengambil screenshot login screen yang muncul, saya membutuhkan bantuan aplikasi ImageMagick, virtual terminal, dan sebuah baris perintah. Baris perintah ini saya sederhanakan dengan membuat sebuah file bash script agar mudah saat pemakaian (tidak mengetik baris perintah yang panjang).

Langkah-langkah yang dilakukan:

  1. JIka aplikasi ImageMagick belum terinstal, maka installah terlebih dahulu. Command: sudo pacman -S imagemagick
  2. Buatlah sebuah file kosong menggunakan text editor (misal vim, nano, atau gedit). Isi file tersebut dengan: su -c ‘chvt 1; sleep 4; import -display  :0.0 -window root /home/alunux/shot1.png; chvt 2;’
  3. Ubah ‘alunux’ dengan username anda. Save file tersebut. Ubahlah permission file tersebut menggunakan chmod. Command: chmod +x file
  4. Sekarang log out.
  5. Pindah ke VT lain (selain VT 1, karena biasanya VT 1 adalah tempat di mana desktop kita berjalan). Jika sudah masuk login lah dengan memasukkan username dan password.
  6. Masuk sebagai root. Command: su -
  7. Masuk tempat di mana kita menyimpan file script yang telah kita buat tadi. Command: cd /direktorinya
  8. Jalankan script tersebut. Command: ./file_scriptnya
  9. Tunggu sekitar 5 detik, kemudian lakukan login seperti biasa.
  10. Buka file manager dan kemudian cari file bernama shot1.png yang terletak di direktori home anda.
  11. Selesai.

[Chapter 2] Instal Aplikasi Tambahan Melalui Terminal

Untuk menginstal suatu atau kumpulan aplikasi saya membutuhkan sebuah package manager. Pacman adalah package manager yang saya pakai untuk menginstal dan mengelola aplikasi yang saya miliki. Pacman digunakan untuk menginstal dari repositori resmi Arch Linux. Untuk paket aplikasi dari AUR (Arch User Repository) saya membutuhkan package manager Pacaur.

[Chapter 3] Memasang dan Menjalankan Aplikasi Line.

Untuk  menginstal dan menjalankan aplikasi LINE (dan aplikasi Windows lainnya) saya membutuhkan paket aplikasi Wine. Penginstalan Wine saya lakukan seperti metode yang ada di Chapter 2. Setelah Wine terinstal, buka Wine Configuration (winecfg). Setting modenya menjadi Windows 7. Terakhir melakukan instalasi aplikasi LINE.

LINE berhasil terinstal dan berjalan dengan baik (Klik untuk memperbesar)

Tulisan dikerjakan menggunakan WordPress dengan bantuan Mozilla Firefox 32.0.1. Sistem operasi yang digunakan yaitu Arch Linux dengan Desktop Environment GNOME 3.12.2. Shell menggunakan Bash. Wine yang digunakan versi 1.7.26. Semua screenshot yang ada di tulisan ini diambil dengan bantuan Shutter 0.93.
