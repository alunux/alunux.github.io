Tampilan LibreOffice Di Fedora 24

@alunux · Aug 18, 2016 · 1 min read

Bismillahirrahmanirrahim.

Sejak saat pertama kali saya melakukan pembaruan dari Fedora 23 ke Fedora 24, saya kurang suka dengan tampilan LibreOffice-nya yang memakai GTK3 secara default. Di Fedora 24, paket libreoffice-gtk3 merupakan paket standar dari aplikasi LibreOffice dan merupakan dependensi dari paket libreoffice-core. Jika kita menghapus paket libreoffice-gtk3, maka paket-paket LibreOffice lainnya juga akan ikut terhapus.

Nah, di sini saya ingin LibreOffice-nya memakai GTK2 secara default. Harapannya tampilan LibreOffice akan lebih baik dari sebelumnya. Cukup jalankan berintah berikut:

sudo sed -i s/Exec=/Exec=env\ SAL_USE_VCLPLUGIN=gtk\ /g /usr/share/applications/libreoffice-*

Kemudian logout dari sesi desktop, login kembali, dan coba jalankan salah satu aplikasi LibreOffice untuk melihat hasilnya. Oh iya, perintah di atas perlu dijalankan kembali jika paket-paket LibreOffice mendapatkan pembaruan.
