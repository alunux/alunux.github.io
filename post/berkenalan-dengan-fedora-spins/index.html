Berkenalan Dengan Fedora Spins

@alunux · Jul 20, 2014 · 4 min read

Apakah spin itu? Fedora spin adalah versi alternatif dari Fedora, disesuaikan untuk berbagai jenis pengguna dengan memilih aplikasinya sendiri dan kostumisasi yang lain. Spin pertama tersedia bersamaan dengan rilis Fedora 7 pada Mei 2007. Proses pembuatan Spin dan perbaikan Spin dikelola oleh Fedora Spins SIG. (Sumber: http://spins.fedoraproject.org/id/about)

Kalau Anda adalah pengguna Ubuntu maka Anda pastinya tidak asing dengan berbagai varian Ubuntu, seperti Kubuntu, Ubuntu GNOME, Xubuntu, Lubuntu, dan Edubuntu.  Fedora Spins diciptakan untuk memudahkan  pekerjaan dengan seseorang dengan kebutuhan spesifik. Misal Anda adalah seorang mahasiswa Teknik Elektro, Fedora Spins menawarkan kepada Anda Fedora Electronic Lab (FEL). Fedora ini menawarkan kumpulan software yang berhubungan dengan elektronika. Mulai dari desain perangkat keras hingga simulasi.

Berdasarkan website Fedora Spins (http://spins.fedoraproject.org/), saat ini total jumlah ada sebanyak 12 Spins yang tersedia. Saya akan membaginya menjadi dua kategori.

  1. Edisi Desktop Environment dan
  2. Edisi Kebutuhan Spesifik.

Pembahasan 1: Edisi Desktop Environment

Fedora merupakan distro linux yang memakai GNOME sebagai desktop environment defaultnya. Adapaun Fedora dengan desktop environment lain tersedia dalam bentuk Spins. Berikut Daftar dan penjelasan singkat dari masing-masing Spins Edisi Desktop Environment.

  1. Fedora KDE Plasma Desktop Edition

    Fedora KDE Plasma Desktop Edition adalah Fedora yang memakai KDE sebagai desktop environmentnya. Kumpulan aplikasi KDE ikut juga di dalamnya. Website: http://spins.fedoraproject.org/id/kde/#home

  2. Fedora Xfce Desktop

    Fedora Xfce merupakan Fedora yang memakai XFCE sebagai desktop environmentnya. Website: http://spins.fedoraproject.org/id/xfce/#home

  3. Fedora LXDE Desktop

    Fedora LXDE merupakan Fedora yang memakai LXDE sebagai desktop environmentnya. Website: http://spins.fedoraproject.org/id/lxde/#home

  4. Fedora MATE-Compiz

    Fedora MATE-Compiz merupakan Fedora yang memakai MATE sebagai desktop environmentnya dan disertakan Compiz Fusion sebagai window managernya. Jika Anda ingin bernostalgia dengan GNOME 2 yang dibubuhi efek 3D, maka Fedora MATE-Compiz merupakan distro linux yang bisa anda coba. Website: http://spins.fedoraproject.org/id/mate-compiz/

Pembahasan 2: Edisi Kebutuhan Spesifik

  1. Fedora Security Lab

    Fedora Security Lab menyediakan lingkungan pengujian yang aman untuk bekerja pada audit keamanan, forensik, sistem penyelamatan dan pengajaran metodologi pengujian keamanan di universitas-universitas dan organisasi lainnya. Ia dilengkapi dengan XFCE sebagai desktop environmentnya dan fitur live image yang memungkinkan Fedora Security Lab dijalankan secara live menggunakan usb flash disk atau media instalasi lainnya.  Website: http://spins.fedoraproject.org/id/security/

  2. Fedora Electronic Lab

    Fedora Electronic Lab menyediakan berbagai software dan tool untuk desain perangkat keras dan simulasi. Fedora Electronic Lab sangat cocok bagi anda yang menggeluti bidang elektronika. Kumpulan software open source Electronic Design Automation (EDA) pilihan telah dibundle di dalamnya. Ini bisa menjadi solusi bagi Anda yang mempunyai masalah keuangan untuk membeli lisensi software elektronika. Fedora Electronic Lab sangat cocok bagi pelajar dan mahasiswa yang ingin mempelajari elektronika tetapi terkendala di biaya. Website: http://spins.fedoraproject.org/id/electronic-lab/

  3. Fedora Design Suite

    Fedora Design Suite menyediakan berbagai software yang dibutuhkan bagi mereka yang bergelut di bidang desain grafis. Mulai dari fotografi, desain web, antarmuka sebuah aplikasi, pemodelan 3D, hingga pembuatan animasi. Software terkenal seperti Blender, GIMP, dan Inkscape sudah tersedia di dalamnya. Website: http://spins.fedoraproject.org/id/design/#home

  4. Fedora Scientific

    Fedora Scientific menyediakan berbagai software dan pustaka untuk keperluan perhitungan matematis, statistik, dan pengembangan software. Sekumpulan IDE dan pustaka untuk berbagai macam bahasa pemrograman telah ada di dalamnya. Fedora Scientific memakai KDE sebagai desktop environmentnya. Website: http://spins.fedoraproject.org/id/scientific-kde/

  5. Fedora Sugar on a Stick

    Fedora Sugar on a Stick menyediakan lingkungan yang menyenangkan bagi anak-anak sebagai media pembelajaran bagi mereka. Mulai dari membaca ebook, menulis, menggambar, dan lain sebagainya. Saya tidak terlalu mengetahui Spin ini. Silahkan menuju ke websitenya untuk mendapatkan informasi lebih lanjut. Website: http://spins.fedoraproject.org/id/soas/#home

  6. Fedora Games

    Fedora Games menyediakan lingkungan yang menyenangkan bagi para gamer. Mengapa? Karena Ia mempunyai kumpulan game open source terbaik dan popular serta lebih dari 100  game lainnya. Website: http://spins.fedoraproject.org/id/games/

  7. Fedora Jam

    Fedora Jam menyediakan berbagai software audio untuk keperluan membuat audio, melakukan editing, dan rekayasa audio lainnya. Sangat cocok bagi anda yang menyukai audio editing. Website: http://spins.fedoraproject.org/id/jam-kde/

  8. Fedora Robotics Suite

    Fedora Robotics Suite menyediakan lingkungan yang menyenangkan bagi para pengembang robot. Berbagai software open source untuk robotik telah ada di dalamnya. Website: http://spins.fedoraproject.org/id/robotics/

Itu saja perkenalan singkat dengan Fedora Spins. Semoga tulisan ini bermanfaat bagi anda yang ingin memakai linux dengan kebutuhan tertentu.
