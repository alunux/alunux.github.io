Install, Uninstall, Dan Update Software Di Fedora

@alunux · Oct 31, 2014 · 3 min read

Bismillahirrahmanirrahim.

[Chapter 1] Install Paket atau Software

Menginstal paket di Fedora sangatlah mudah untuk dilakukan. Jika kita mengetahui nama suatu paket dengan benar, maka cukup dengan mengetikkan perintah yum install namapaket dalam mode root. Di sini saya juga akan menjelaskan cara mencari suatu paket berdasarkan sedikit informasi tentang paket yang diinginkan dan melihat deskripsi suatu paket.

  • Jika kita memiliki informasi mengenai suatu paket yang kita tidak tahu namanya, maka kita dapat memakai perintah yum search namapaket atau yum search infopaket atau yum search info1 | grep info2.

  • Kita juga dapat melihat deskripsi suatu paket. Untuk melakukan hal tersebut, kita dapat memakai perintah yum info namapaket. Mulai dari nama, arksitektur, versi, ukuran, bahkan url website resmi paket tersebut juga ditampilkan. Banyak informasi berguna yang akan ditampilkan oleh perintah tersebut.

[Chapter 2] Uninstall Paket atau Software

Menghapus suatu paket atau software sama mudahnya ketika kita menginstall paket. Perintahnya pun cukup simpel, yum remove namapaket. Yang perlu diketahui adalah perintah tersebut cuman menghapus paket tersebut saja, paket-paket dependensi yang ikut bersama paket tersebut saat instalasi tidaklah terhapus. Untuk menghapus paket-paket yang tidak diperlukan oleh suatu aplikasi atau sistem, saya biasanya memakai perintah yum autoremove.

[Chapter 3] Update Software dan System

  • Untuk melakukan update sistem Fedora beserta paket-paket yang ada di dalamnya, kita dapat memakai perintah yum update. Jika ada suatu paket yang kita tidak ingin masuk dalam daftar update, maka kita dapat menambahkan --exclude=namapaket. Kita juga dapat menggunakan wildcard. Misal saya tidak ingin kernel saya ikut terupdate, maka perintah yang saya gunakan adalah yum update --exclude=kernel*

  • Jika kita cuman ingin mengupdate suatu paket saja, maka kita dapat memakai perintah yum update namapaket atau yum update paket1 paket2 paket3 dan seterusnya kalau lebih dari satu paket. Misal saya ingin mengupdate firefox, maka perintahnya adalah yum update firefox.

  • Ada suatu kasus di mana Fedora yang kita miliki memiliki paket-paket yang bentrok satu sama lain. Saya pernah mengalami kasus seperti ini. Hal ini terjadi ketika ada paket-paket yang tersedia direpo resmi tapi tersedia juga direpo tambahan kita (unofficial repo). Waktu itu saya ingin menginstall Wine. Tiba-tiba muncul pesan error, Error: Multilib version problems found. Nah, akhirnya saya memakai perintah yum distro-sync atau bisa juga yum distribution-synchronization. Hasilnya, ada beberapa paket yang harus di downgrade agar semua proses manajemen paket berjalan dengan semestinya. Untuk lebih lengkapnya silahkan lihat catatan saya di sini http://pastebin.com/ZFYs1JHD
